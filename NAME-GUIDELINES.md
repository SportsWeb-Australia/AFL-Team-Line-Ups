# Player names — display guidelines

How names should read on the line-up graphic. The goal is a sheet that looks
consistent and professional, and stays readable on a phone.

## Recommended: first initial + surname ("J. Smith")

Use the broadcast convention — **first initial, full stop, surname**:

> J. Smith · D. Matthews · S. McHenry

Why it's the default we recommend:

- It's instantly recognisable as a proper team list (it's how the pros do it).
- It's short, so it fits the on-field plate cleanly on mobile.
- It disambiguates better than a nickname (everyone knows who "J. Smith" is).

## Be consistent — pick ONE style per team

The cardinal rule: **don't mix styles within a single line-up.** A sheet that reads
"Dan Mac, Ando, Whitey, Overman, J. Hayes" looks untidy because it's switching
between nickname, surname and initial-surname. Choose one and apply it to all 22.

In order of preference:

1. **First initial + surname** — `J. Smith` *(recommended)*
2. **Surname only** — `Smith`
3. **First name only** — `Jack`
4. **Club nickname** — `Jonesy` *(fine for Masters/social sides, but commit to it
   for the whole team)*

## Handling duplicates

Two players with the same surname? Disambiguate:

- Two initials: `J. Smith` / `T. Smith`
- Or flip to first-name-led: `Jack S.` / `Tom S.`

## Length & formatting

- **Keep it short — aim for ≤ 14 characters.** On the oval the name wraps to a
  maximum of two lines on mobile and then trims with "…", so long names get cut.
  `J. Smith` never has this problem.
- **Title Case**: `O'Brien`, `Van Der Berg` (watch the length on double-barrels).
- Apostrophes and hyphens are fine.

## Guernsey numbers

Numbers are **optional**. If your side doesn't run numbers (common in Masters /
Over-55s), just leave the number blank — the plate now drops the number entirely
and shows a clean name bar with a small club-colour tab, instead of a row of "–"
placeholders. If you *do* use numbers, enter the guernsey number per player.

## Where you set this in the app

Each player's shown name is the **display name** field in the squad list. Set it
once per player to the style you've chosen, and the same name flows onto the oval,
the bench and the exported PNG.
